RadialProgress

RadialProgress class

A stateless component for rendering a radial progress indicator.

Allows customization of progress value, colors, size, and thickness.

Example:

RadialProgress(
  value: 75,
  color: TextColor.primary,
  size: '4rem',
  thickness: '4px'
)
