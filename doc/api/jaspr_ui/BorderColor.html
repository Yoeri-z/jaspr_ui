BorderColor

BorderColor enum

Border colors for styling component borders.

Provides a set of predefined border color classes that can be used throughout the application for consistent border styling. Each enum value maps to a specific CSS border color class.

Example usage:

final borderColor = BorderColor.inherit;
print(borderColor.className); // Outputs: 'border-inherit'
