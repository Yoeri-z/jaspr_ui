Stat

Stat class

A component that displays a single statistic with a title, value, and description.

The Stat component provides a structured way to display statistical information with customizable colors, optional icons, and alignment options.

Example usage:

Stat(
  title: 'Downloads',
  value: '31K',
  desc: 'Jan 1st - Feb 1st',
  valueColor: TextColor.primary,
  isCenter: true,
  icon: 'some-icon-class'
)
