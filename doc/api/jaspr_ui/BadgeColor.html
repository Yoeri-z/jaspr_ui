BadgeColor

BadgeColor enum

Badge colors for styling components.

This enum provides a standardized set of colors for badge components in the application. Each value corresponds to a specific CSS class that applies the respective color styling.

Example usage:

final badge = BadgeColor.success;
print(badge.className); // Outputs: 'badge-success'
